In this, 6 divs with 5 different colors are implemented. It has 6 different images. Keeping first div fixed, we make all other divs position as absolute. Using JQuery when we scroll, onScroll event is handled. It sets Top for the next div, using current divs coordinates

Formula used is : New div top = base div top - current div top;
It means it calculates by how much is the topmost div scrolled and adds that to the new div. Hence we see all divs are connected.
